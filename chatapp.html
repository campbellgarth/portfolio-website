Chat App

Chat is an RTA chatting app made in React Native. Users are able to chat in real-time, send photos, and send their location.

See the Github Repository

Key Features

Users are able to:

  • Enter their name and choose a background color for the chat screen before joining the chat.
  • See a page that displays the conversation, as well as an input field and submit button.
  • Send images from their gallery
  • Send a photo taken from their camera
  • Send their location
  • Use the app online or offline

Technologies used

  • React Native
  • Gifted Chat
  • Expo
  • Android Studio
  • Google Firebase/Firestore
