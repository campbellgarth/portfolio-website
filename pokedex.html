Pokedex App

The Pokedex App is a JavaScript app that allows you to view information about Generation 1 Pokemon and choose your favorites.

See it liveSee the Github Repository

Key Features

  • App incorporates data from the external PokeAPI
  • Users are able to view a list of Pokemon
  • Upon clicking on a Pokemon, users are able to open a modal with key Pokemon information
  • Users are able to add their favorites to a separate list

Technologies used

  • HTML
  • CSS
  • JavaScript
  • JQuery
  • Bootstrap
  • PokeAPI
